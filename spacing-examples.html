Semantic Spacing Tokens

Purpose-driven spacing for consistent patterns

  • --space-inline (8px) - For inline element gaps like buttons and tags
  • --space-stack (16px) - For vertically stacked related elements
  • --space-inset (16px) - For internal container padding
  • --space-section (128px) - For major section separation
  • --space-page (192px) - For page-level spacing
